# SpatialGuide-Bot
Robot diseñado para asistir a personas, especialmente a aquellas con discapacidad visual, en la navegación y comprensión de entornos interiores controlados.  El sistema se basa en una integración de tecnologías de voz (Whisper), visión multimodal (Gemini API), y navegación inercial (MPU6050 y control PID)
